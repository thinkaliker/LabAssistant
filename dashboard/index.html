Add Host

      

        
In Slice 1, enroll a host from the CLI:

        
manager enroll --name myhost --ip 10.0.0.5 --addr manager:8443

        
Copy the bundle to the host and run associate --bundle bundle.json. The SSH-based Add Host flow arrives in Slice 2.
